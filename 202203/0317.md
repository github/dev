# 实验室管理系统业务功能

## 实验室管理

### 实验室信息

- 名称
- 位置
- 管理员
- 主要业务
- 看板信息

	- 仪器信息
	- 样品信息
	- 试剂耗材信息

### 预约管理（审批流）

- 预约信息

	- 用途
	- 使用时间
	- 操作管理

		- 提前结束
		- 预约前/后操作
		- 取消预约

- 预约台账

	- 设备编号
	- 采购金额
	- 数量
	- 单位
	- 所属实验室
	- 摆放位置
	- 用途备注
	- 使用说明书
	- 状态
	- 下一次维保日期
	- 出厂时间
	- 采购时间
	- 提前告警条件

- 仪器面板

	- 使用记录
	- 维保记录
	- 预约台账

## 样品管理

### 采样抽样

- 规范管理
- 送样管理

	- 样品信息录入

		- 样品名称
		- 采样类型

			- 送样
			- 采样

				- 采样小组管理
				- 采样设备/耗材提醒

		- 送样人/公司
		- 送样时间
		- 采样人
		- 采样时间
		- 采样地点
		- 采样标准
		- 样品是否合格
		- 样品编号（一份样品一个编号，系统内唯一）
		- 有效期

			- 到期提醒设置

	- 样品分样（制样，留样等）

		- 样品存储位置
		- 留样时效

			- 到期提醒设置

### 样品存储

- 样品存储入库记录
- 样品报废管理
- 样品过期提醒

### 样品流转

- 样品流转记录

	- 流转信息

		- 检验任务
		- 检验人
		- 样品领取

			- 领取时间
			- 领取人
			- 领取数量

		- 样品归还

			- 归还时间
			- 归还数量
			- 归还人

		- 用途

### 样品盘点

### 留样管理

- 留样信息

	- 留样入库

		- 存储位置
		- 样品编号

	- 留样提醒

- 留样报废

## 试剂耗材管理

### 耗材档案管理

- 耗材分类管理
- 耗材档案信息

	- 品名
	- 规格
	- 级别
	- 数量&单位
	- 采购时效
	- 证书等质量证明文件
	- 验收人
	- 验收时间
	- 低库存预警

		- 预警条件设置

	- 耗材过期预警

		- 预警条件设置

	- 存储位置

### 采购管理

- 采购计划
- 试试采购
- 耗材验收

### 耗材领用（审批流）

- 耗材领用申请
- 出库记录
- 采购入库记录
- 归还退库记录

## 检验任务管理

### 检验类别管理

- 检验类别

	- 名称
	- 类型
	- 样品方式

		- 送样
		- 采样

	- 检验项目

		- 检验项目管理

			- 项目名称
			- 合格条件
			- 检测费用
			- 检验标准

		- 检验标准

	- 检验费用范围
	- 检验报告模板

### 接收任务

- 委托任务

	- 项目名称
	- 检验类别
	- 委托单位/受检单位

		- 名称
		- 联系人
		- 联系方式

	- 检验样品
	- 任务接收时间
	- 对接人

### 合同管理

### 下发任务

- 检验项目组

	- 项目组名
	- 负责人
	- 负责业务
	- 标签
	- 状态

		- 是否承接任务中
		- 主要人员状态

	- 看板

		- 检验任务统计
		- 标签
		- 简介

- 下发任务管理

	- 分配检验项目组
	- 检验状态
	- 检验结果
	- 检验报告

## 检验报告管理

### 检验结果

- 根据任务检索
- 根据项目组检索
- 检验结果信息

	- 检验数据
	- 检验结论

### 检验报告（审批流）

- 任务检查结果
- 检验报告

	- 检验任务
	- 检验人员
	- 检验时间
	- 检验地点
	- 检验结果
	- 检验项目数据&结果
	- 检验结果仅对样品或批次有效声明
	- 编制人员
	- 审核人员
	- 批准人

- 检验报告导出/打印
